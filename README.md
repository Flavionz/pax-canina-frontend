# Projet Fil Rouge Club Canine

## Description du Projet

Le **Club Canine** est une application web conçue pour faciliter la gestion des cours, des inscriptions et des interactions entre les propriétaires de chiens, les entraîneurs et les responsables du club. L'application offre une interface conviviale pour s'inscrire, réserver des cours, gérer les profils des chiens et suivre les progrès.

### Fonctionnalités Principales

- **Inscription des Propriétaires et des Chiens** : Les propriétaires peuvent s'inscrire et ajouter leurs chiens.
- **Gestion des Cours** : Les responsables peuvent créer et gérer des cours et des sessions.
- **Réservation de Cours** : Les propriétaires peuvent réserver des cours pour leurs chiens.
- **Tableau de Bord** : Les entraîneurs et les responsables ont accès à un tableau de bord pour suivre les inscriptions et les sessions.
- **Notifications** : Les utilisateurs reçoivent des notifications pour les rappels et les mises à jour.

## Technologies Utilisées

- **Frontend** : Angular
- **Backend** : Spring Boot
- **Base de Données** : MySQL
- **ORM** : Hibernate (Spring Data JPA)
- **Autres Outils** : Git, Maven, Docker (optionnel)
